数据探索与挑战 (EDA)

细粒度类别构成

DERM12345 数据集包含 40 个子类，映射为 5 个 Super-Class。分析显示各类别内部成分复杂（如图所示），且存在严重的长尾分布。Class 0 (良性痣) 样本量过万，而部分恶性类别样本稀缺，这对模型的特征提取能力提出了极高要求。

特征空间重叠挑战

Zero-shot UMAP 可视化揭示了关键问题：虽然模型能区分黑色素与非黑色素病变，但良性 (Class 0) 与恶性黑色素瘤 (Class 1) 在特征空间中高度纠缠。这证明了仅靠线性探测是不够的，必须进行全参数微调以重构特征流形，拉大良恶性样本的距离。

模型架构与评估

采用 ViT-Large 骨干网络，结合 Weighted Random Sampling 策略，在 NVIDIA GPU 上进行了 20 Epochs 的全参数微调。

混淆矩阵分析

混淆矩阵显示，模型在 Class 0 (良性) 和 Class 4 (非黑恶性) 上表现优异（对角线颜色深）。

尽管样本稀缺，通过加权采样策略，模型在最关键的 Class 1 (黑色素瘤) 上的识别能力得到了显著保障。虽然仍有部分误判为良性，但相比线性探测阶段，召回率已有质的提升。

cmd = (

2. Streamlit 部署 (Mac MPS 加速适配) 
@st.cache_resource
def load_model_mac():
    # 自动适配 Apple Silicon (M1/M2/M3)
    device = torch.device("mps") if torch.backends.mps.is_available() else torch.device("cpu")
    
    # 手动构建 5 分类模型结构 (显式开启 Mean Pooling)
    from models.modeling_finetune import panderm_large_patch16_224
    model = panderm_large_patch16_224(num_classes=5, use_mean_pooling=True)
    
    # 加载微调权重 (处理 DDP 前缀)
    checkpoint = torch.load(MODEL_PATH, map_location='cpu')
    new_state_dict = {k.replace("module.", ""): v for k, v in checkpoint['model'].items()}
    model.load_state_dict(new_state_dict, strict=False)
    
    return model.to(device).eval()
