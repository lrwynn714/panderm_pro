数据探索与挑战 (EDA)

极端类别不平衡

数据集呈现严重的长尾分布。Class 0 (良性痣) 样本量过万，而 Class 3 (不确定病变) 仅有数十例。这导致标准交叉熵损失函数失效，我们在训练中引入了 Weighted Random Sampler。

特征空间重叠

Zero-shot UMAP 可视化显示，良性与恶性黑色素瘤在特征空间中高度纠缠。这证明了仅靠线性探测是不够的，必须进行全参数微调以重构特征流形。

@st.cache_resource
def load_model_mac():
    # 自动适配 Apple Silicon
    device = torch.device("mps") if torch.backends.mps.is_available() else torch.device("cpu")
    # 手动构建 5 分类模型结构
